## ‚ú® Features

- **User Accounts**: Sign up and login to your own personalized account.
- **Create Quizzes**: Build custom quizzes with unlimited questions.
- **Interactive Gameplay**:
  - Immediate visual feedback (Green for Correct, Red for Wrong).
  - Score percentage and motivational messages.
- **Management**: Delete quizzes you created directly from the dashboard.
- **Responsive Design**: Clean, modern UI that works on desktop and mobile.

## üöÄ Getting Started

### Prerequisites
You just need a modern web browser (Chrome, Firefox, Edge, etc.).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/QuizIt.git
   ```
2. Open `index.html` in your browser.

**Note**: Since this project uses `sessionStorage` and `localStorage`, handling file protocols (`file://`) might sometimes be strict in some browsers, but generally works. For best experience, run a local live server (e.g., using VS Code Live Server extension).
